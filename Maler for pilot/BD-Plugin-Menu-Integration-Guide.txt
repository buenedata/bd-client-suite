================================================================================
BD PLUGIN MENU INTEGRATION GUIDE
================================================================================
BUENE DATA - Felles WordPress Admin Menu Structure
Versjon: 2.0
Dato: 21. juli 2025

================================================================================
OVERSIKT
================================================================================

Denne guiden beskriver hvordan alle BD (Buene Data) plugins skal integreres
under en felles "Buene Data" hovedmeny i WordPress admin. Dette sikrer en
konsistent brukeropplevelse og god organisering av alle BD-produkter.

NYE FUNKSJONER V2.0:
- Dynamisk plugin registrering system
- Automatisk oppdatering av oversiktsside
- Plugin status deteksjon (aktiv/inaktiv/ikke installert)
- Auto-oppdagende system for BD plugins

================================================================================
IMPLEMENTERING
================================================================================

1. PLUGIN REGISTRERING (ANBEFALT METODE)
----------------------------------------

Fra versjon 2.0 har BD CleanDash et dynamisk plugin registreringssystem.
Registrer din plugin for optimal integrasjon:

```php
// Legg til i din plugin's main klasse eller init funksjon
add_action('bd_plugin_registry_init', function($registry) {
    $registry->register_plugin(array(
        'slug' => 'bd-your-plugin',
        'name' => 'BD Your Plugin',
        'description' => 'Kort beskrivelse av hva din plugin gjør.',
        'icon' => '🔧',
        'version' => YOUR_PLUGIN_VERSION,
        'status' => 'auto', // Auto-detekterer status
        'admin_url' => admin_url('admin.php?page=bd-your-plugin'), // Hovedadministrasjonsside
        'settings_url' => admin_url('admin.php?page=bd-your-plugin-settings'), // Innstillingsside (valgfritt)
        'plugin_file' => plugin_basename(__FILE__),
        'download_url' => 'https://buenedata.no/downloads/bd-your-plugin.zip', // Valgfritt
        'wordpress_org_url' => 'https://wordpress.org/plugins/bd-your-plugin/', // Valgfritt
        'docs_url' => 'https://buenedata.no/docs/your-plugin/', // Valgfritt
        'support_url' => 'https://buenedata.no/support/' // Valgfritt
    ));
});
```

2. HOVEDMENY SJEKK OG OPPRETTELSE
---------------------------------

Legg til følgende kode i din plugin's admin interface klasse:

```php
public function add_admin_menu() {
    // Sjekk om BD hovedmeny allerede eksisterer
    global $menu;
    $bd_menu_exists = false;
    foreach ($menu as $menu_item) {
        if (isset($menu_item[2]) && $menu_item[2] === 'buene-data') {
            $bd_menu_exists = true;
            break;
        }
    }
    
    // Hvis BD hovedmeny ikke eksisterer, opprett den
    if (!$bd_menu_exists) {
        add_menu_page(
            __('Buene Data', 'your-text-domain'),
            __('Buene Data', 'your-text-domain'),
            'manage_options',
            'buene-data',
            [$this, 'render_bd_overview'],
            'data:image/svg+xml;base64,' . base64_encode('<svg width="20" height="20" viewBox="0 0 20 20" fill="none" xmlns="http://www.w3.org/2000/svg"><path d="M10 2L3 7V18H7V14H13V18H17V7L10 2Z" fill="currentColor"/></svg>'),
            58.5
        );
    }
    
    // Legg til din plugin som submeny
    add_submenu_page(
        'buene-data',
        __('BD Your Plugin', 'your-text-domain'),
        __('🔧 Your Plugin', 'your-text-domain'),
        'manage_options',
        'bd-your-plugin',
        [$this, 'render_admin_page']
    );
}
```

3. AUTO-DETEKSJON SYSTEM (FALLBACK)
-----------------------------------

Hvis du ikke registrerer din plugin manuelt, vil BD CleanDash forsøke å
auto-detektere den basert på følgende kriterier:

- Plugin navn starter med "BD " eller inneholder "Buene Data"
- Plugin beskrivelse inneholder "Buene Data" eller "BD Plugin"
- Plugin forfatter er "Buene Data" eller forfatter URI inneholder "buenedata.no"

For optimal resultat, anbefales manuell registrering (metode 1).

PLUGIN REGISTRERING PARAMETERE:
------------------------------

| Parameter         | Type   | Påkrevd | Beskrivelse                              |
|-------------------|--------|---------|------------------------------------------|
| slug              | string | Ja      | Unik plugin identifikator                |
| name              | string | Ja      | Visningsnavn for plugin                  |
| description       | string | Ja      | Kort beskrivelse for oversiktskortet     |
| icon              | string | Nei     | Emoji eller ikon (standard: 🔌)         |
| version           | string | Nei     | Plugin versjonsnummer                    |
| status            | string | Nei     | auto/active/inactive/not-installed       |
| admin_url         | string | Nei     | URL til plugin hovedadministrasjonsside |
| settings_url      | string | Nei     | URL til plugin innstillingsside         |
| plugin_file       | string | Nei     | Plugin fil sti (for status deteksjon)   |
| download_url      | string | Nei     | URL for direkte nedlasting               |
| wordpress_org_url | string | Nei     | URL til WordPress.org plugin side       |
| docs_url          | string | Nei     | URL til dokumentasjon                    |
| support_url       | string | Nei     | URL til support                          |

KNAPP-LOGIKK FOR AKTIVE PLUGINS:
- Hvis 'admin_url' er satt: "Konfigurer" → leder til hovedsiden
- Hvis 'settings_url' er satt: "Innstillinger" → leder til innstillinger
- Hvis 'docs_url' er satt: "Dokumentasjon" → leder til dokumentasjon
- Ellers: "Vis detaljer" → leder til plugins.php

KNAPP-LOGIKK FOR IKKE-INSTALLERTE PLUGINS:
- Hvis 'download_url' er satt: "Last ned" → direkte nedlasting
- Hvis 'wordpress_org_url' er satt: "Installer fra WordPress.org"
- Hvis 'docs_url' er satt: "Les mer" → dokumentasjon
- Ellers: "Kommer snart" (deaktivert knapp)

Her er et komplett eksempel på en BD plugin med full integrasjon:

```php
<?php
/**
 * BD Example Plugin
 */

class BD_Example_Plugin {
    
    public function __construct() {
        add_action('init', array($this, 'init'));
        add_action('bd_plugin_registry_init', array($this, 'register_with_bd_system'));
        add_action('admin_menu', array($this, 'add_admin_menu'));
    }
    
    public function init() {
        // Plugin initialisering
    }
    
    /**
     * Registrer med BD systemet
     */
    public function register_with_bd_system($registry) {
        $registry->register_plugin(array(
            'slug' => 'bd-example-plugin',
            'name' => 'BD Example Plugin',
            'description' => 'Eksempel plugin som viser BD integrasjon.',
            'icon' => '🔧',
            'version' => '1.0.0',
            'status' => 'auto',
            'admin_url' => admin_url('admin.php?page=bd-example-plugin'),
            'settings_url' => admin_url('admin.php?page=bd-example-plugin-settings'),
            'plugin_file' => plugin_basename(__FILE__),
            'download_url' => 'https://buenedata.no/downloads/bd-example-plugin.zip',
            'docs_url' => 'https://buenedata.no/docs/example-plugin/',
            'support_url' => 'https://buenedata.no/support/'
        ));
    }
    
    /**
     * Legg til admin menu
     */
    public function add_admin_menu() {
        // Sjekk om BD hovedmeny eksisterer
        $bd_menu_exists = false;
        global $menu;
        if (is_array($menu)) {
            foreach ($menu as $menu_item) {
                if (isset($menu_item[2]) && $menu_item[2] === 'buene-data') {
                    $bd_menu_exists = true;
                    break;
                }
            }
        }
        
        // Opprett BD hovedmeny hvis den ikke eksisterer
        if (!$bd_menu_exists) {
            add_menu_page(
                __('Buene Data', 'bd-example-plugin'),
                __('Buene Data', 'bd-example-plugin'),
                'manage_options',
                'buene-data',
                array($this, 'render_overview_fallback'),
                'data:image/svg+xml;base64,' . base64_encode('<svg width="20" height="20" viewBox="0 0 20 20" fill="none" xmlns="http://www.w3.org/2000/svg"><path d="M10 2L3 7V18H7V14H13V18H17V7L10 2Z" fill="currentColor"/></svg>'),
                58.5
            );
        }
        
        // Legg til denne plugin som submenu
        add_submenu_page(
            'buene-data',
            __('BD Example Plugin', 'bd-example-plugin'),
            __('🔧 Example Plugin', 'bd-example-plugin'),
            'manage_options',
            'bd-example-plugin',
            array($this, 'render_admin_page')
        );
    }
    
    /**
     * Fallback oversiktsside
     */
    public function render_overview_fallback() {
        if (class_exists('BD_Plugin_Registry')) {
            wp_redirect(admin_url('admin.php?page=buene-data'));
            exit;
        }
        
        echo '<div class="wrap">';
        echo '<h1>' . __('Buene Data Plugins', 'bd-example-plugin') . '</h1>';
        echo '<p>' . __('Installer BD CleanDash for bedre oversikt.', 'bd-example-plugin') . '</p>';
        echo '</div>';
    }
    
    /**
     * Plugin admin side
     */
    public function render_admin_page() {
        echo '<div class="wrap">';
        echo '<h1>' . __('BD Example Plugin', 'bd-example-plugin') . '</h1>';
        echo '<p>' . __('Dette er et eksempel på BD plugin integrasjon.', 'bd-example-plugin') . '</p>';
        echo '</div>';
    }
}

new BD_Example_Plugin();
```

8. PLUGIN KNAPP OPPFØRSEL
------------------------

BD systemet velger automatisk riktig knapp tekst og URL basert på plugin status:

AKTIVE PLUGINS:
1. Prioritert rekkefølge for knappevalg:
   - admin_url → "Konfigurer" 
   - settings_url → "Innstillinger"
   - docs_url → "Dokumentasjon"
   - Fallback → "Vis detaljer" (leder til plugins.php)

INAKTIVE PLUGINS:
- Hvis plugin_file er tilgjengelig → "Aktiver" (automatisk aktivering)
- Ellers → "Gå til plugins" (leder til plugins.php)

IKKE-INSTALLERTE PLUGINS:
1. Prioritert rekkefølge for knappevalg:
   - download_url → "Last ned"
   - wordpress_org_url → "Installer fra WordPress.org"
   - docs_url → "Les mer"
   - Fallback → "Kommer snart" (deaktivert)

KOMMER SNART PLUGINS:
- Alltid "Kommer snart" (deaktivert knapp)

ANBEFALING FOR PLUGIN UTVIKLERE:
- Sett alltid 'admin_url' for hovedadministrasjonsside
- Bruk 'settings_url' hvis du har en dedikert innstillingsside
- Inkluder 'docs_url' for hjelp når plugin ikke har admin interface
- Angi 'plugin_file' for korrekt aktivering/deaktivering
- Legg til 'download_url' eller 'wordpress_org_url' for enkelt installasjon

================================================================================
BESTE PRAKSIS FOR PLUGIN DESIGN
================================================================================

1. PLUGIN REGISTRERING
   - Bruk alltid plugin registreringssystemet for optimal integrasjon
   - Registrer på 'bd_plugin_registry_init' hook
   - Angi 'status' => 'auto' for automatisk status deteksjon

2. MENU POSISJON
   - Hovedmeny skal ha posisjon 58.5 (mellom Comments og Appearance)

3. CAPABILITIES
   - Bruk 'manage_options' for alle BD plugin sider
   - Vurder mer spesifikke capabilities ved behov

4. TEXT DOMAIN
   - Bruk konsistent text domain for alle strenger
   - Følg WordPress i18n beste praksis

5. HOOK PRIORITET
   - Kjør menu-opprettelse på 'admin_menu' hook
   - Kjør plugin registrering på 'bd_plugin_registry_init' hook (prioritet 10 eller lavere)

6. KOMPATIBILITET
   - Sjekk alltid om hovedmenyen eksisterer før opprettelse
   - Håndter gracefully hvis andre plugins endrer menu-struktur
   - Sjekk om BD_Plugin_Registry klassen eksisterer før registrering

7. PERFORMANCE
   - Cache menu-eksistens sjekk hvis mulig
   - Unngå tunge operasjoner i menu callbacks

8. PLUGIN METADATA
   - Angi alltid admin_url for enkel navigasjon
   - Bruk beskrivende ikoner (emoji) for visuell konsistens
   - Inkluder dokumentasjon og support URLer når tilgjengelig

EKSEMPEL IMPLEMENTERING

Komplett eksempel for en ny BD plugin:

```php
class BD_YourPlugin_Admin {
    
    public function init() {
        add_action('admin_menu', [$this, 'add_admin_menu']);
        add_action('admin_enqueue_scripts', [$this, 'enqueue_admin_scripts']);
    }
    
    public function add_admin_menu() {
        // Sjekk om BD hovedmeny eksisterer
        global $menu;
        $bd_menu_exists = false;
        foreach ($menu as $menu_item) {
            if (isset($menu_item[0]) && strpos($menu_item[0], 'Buene Data') !== false) {
                $bd_menu_exists = true;
                break;
            }
        }
        
        // Opprett hovedmeny hvis den ikke eksisterer
        if (!$bd_menu_exists) {
            add_menu_page(
                __('Buene Data', 'bd-your-plugin'),
                __('Buene Data', 'bd-your-plugin'),
                'manage_options',
                'buene-data',
                [$this, 'render_bd_overview'],
                'data:image/svg+xml;base64,' . base64_encode('<svg width="20" height="20" viewBox="0 0 20 20" fill="none" xmlns="http://www.w3.org/2000/svg"><path d="M10 2L3 7V18H7V14H13V18H17V7L10 2Z" fill="currentColor"/></svg>'),
                58.5
            );
        }
        
        // Legg til din plugin som submeny
        add_submenu_page(
            'buene-data',
            __('BD YourPlugin', 'bd-your-plugin'),
            __('🔧 YourPlugin', 'bd-your-plugin'),
            'manage_options',
            'bd-your-plugin',
            [$this, 'render_admin_page']
        );
    }
    
    public function enqueue_admin_scripts($hook) {
        if (strpos($hook, 'bd-') === false && strpos($hook, 'buene-data') === false) {
            return;
        }
        
        // Last inn BD felles styling
        wp_enqueue_style('bd-admin-style', plugin_dir_url(__FILE__) . 'assets/bd-admin.css');
    }
    
    public function render_admin_page() {
        ?>
        <div class="wrap bd-your-plugin-wrap">
            <div class="bd-your-plugin-header">
                <h1>🔧 <?php _e('BD YourPlugin', 'bd-your-plugin'); ?></h1>
                <p><?php _e('Beskrivelse av din plugin.', 'bd-your-plugin'); ?></p>
            </div>
            
            <!-- Din plugin innhold her -->
            
        </div>
        <?php
    }
    
    // Implementer render_bd_overview() kun hvis din plugin kan være den første installerte
    public function render_bd_overview() {
        // Se eksempel ovenfor
    }
}
```
